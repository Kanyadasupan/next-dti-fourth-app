# 🧮 เครื่องมือคำนวณออนไลน์ (Online Calculators)
### ตัวช่วยคำนวณสำหรับชีวิตประจำวันของคุณ 📊
**Web Application developed with Next.js & Tailwind CSS**

ยินดีต้อนรับสู่โปรเจกต์เครื่องมือคำนวณออนไลน์ (Online Calculators) ตัวช่วยที่รวบรวมเครื่องมือสำคัญในชีวิตประจำวันไว้ในที่เดียว ทั้งการประเมินสุขภาพเบื้องต้นและการวางแผนการเงิน ออกแบบมาให้ใช้งานง่าย รวดเร็ว และแสดงผลได้สวยงามในทุกหน้าจอ

---

## ✨ คุณสมบัติหลัก (Features)
* **⚖️ BMI Calculation:** คำนวณค่าดัชนีมวลกาย เพื่อประเมินภาวะอ้วนหรือผอม
* **🔥 BMR Calculation:** คำนวณอัตราการเผาผลาญพลังงานพื้นฐานของร่างกายในแต่ละวัน
* **🚗 Car Installment:** คำนวณค่างวดรถยนต์รายเดือน พร้อมฟังก์ชันปรับเปลี่ยนเงินดาวน์และระยะเวลาผ่อน
* **🎨 Simple & Clean UI:** อินเทอร์เฟซเรียบง่าย สะอาดตา ใช้งานสะดวก ไม่ซับซ้อน
* **💻 Responsive Design:** รองรับการทำงานอย่างสมบูรณ์แบบบนทุกอุปกรณ์ ทั้งคอมพิวเตอร์ แท็บเล็ต และสมาร์ทโฟน

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)
* **Framework:** [Next.js](https://nextjs.org/) (React)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Language:** JavaScript / TypeScript

---

## 🚀 การติดตั้งและใช้งาน (Setup & Installation)

1.  **Clone โปรเจกต์:**
    ```bash
    git clone https://github.com/Kanyadasupan/next-dti-fourth-app.git
    cd ชื่อโปรเจกต์
    ```
    *(ตรวจสอบและเปลี่ยน URL ของ Repository ให้ตรงกับชื่อ Repository ของคุณ)*

2.  **ติดตั้ง Dependencies:**
    ```bash
    npm install
    ```

3.  **รันแอปพลิเคชัน:**
    ```bash
    npm run dev
    ```
    *จากนั้นเปิดเบราว์เซอร์และเข้าไปที่ **http://localhost:3000** เพื่อดูผลลัพธ์*
